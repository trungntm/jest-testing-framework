# jest-testing-framework

##### INSTALL JEST USING YARN
- yarn add jest --dev

##### INSTALL JEST USING NPM
- npm install jest -save-dev

##### INSTALL JEST-EXTENDED
- Using npm : npm i --save-dev jest-extended
- Using yarn : yarn add -D jest-extended
